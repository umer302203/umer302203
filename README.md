## 🚀 Featured Projects

### 🔹 RAG-CHATBOT

A Retrieval-Augmented Generation chatbot that answers questions strictly from uploaded PDFs — freezing model weights and grounding responses in a vector database instead of relying on costly fine-tuning.

### 🔹 Multi-model-AI-Assistant

A full-stack GenAI web app built with Flask and LangChain that dynamically switches between Meta Llama 4, IBM Granite, and Mistral in real time, with strict JSON-parsed, hallucination-safe output.
